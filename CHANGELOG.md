# Changelog

## [Pre-Alpha-0.4.1] - 2025-01-XX

### 🔒 Security
- Обновлён Next.js с 15.3.5 до 16.1.7 (исправлена уязвимость CVE-2025-66478)
- Обновлён Swiper с 11.1.14 до 12.1.2 (исправлена критическая уязвимость prototype pollution)
- Исправлены все остальные уязвимости зависимостей

### ✨ Features
- Реализована полная функция создания отзывов в `/reviews/create`
- Добавлена валидация формы создания отзыва
- Добавлены переводы для ошибок валидации (ru, en, uk, be)

### 🔧 Changes
- Мигрирован с `middleware.js` на `proxy.js` (новое соглашение Next.js 16)
- Исправлен синтаксис CSS в `styles/globals.css` для совместимости с Turbopack
- Создан файл `.env.local` с шаблоном переменных окружения
- Добавлена подробная инструкция по настройке в `SETUP_GUIDE.md`

### 🐛 Bug Fixes
- Исправлена конвертация рейтинга с шкалы 0-10 в 1-5
- Исправлено поле `text` вместо `review_text` в API создания отзыва
- Убрана неиспользуемая переменная в обработчике отправки формы
